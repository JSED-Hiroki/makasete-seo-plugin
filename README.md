## Plugin Installation Guide

### Requirements

- WordPress 5.6 or higher
- PHP 8.0 or higher
- A Makasete account at [makasete.app](https://makasete.app/)

### Option 1: Install via GitHub (Manual Upload)

1. Go to the `Releases` page of this repository.
2. Download the latest `makasete-seo.zip` file.
3. In your WordPress admin dashboard, navigate to **Plugins → Add New → Upload Plugin**.
4. Click **Choose File**, select the downloaded ZIP, and click **Install Now**.
5. After installation, click **Activate Plugin**.

### Option 2: Install via FTP / File Manager

1. Download and extract the ZIP file from the `Releases` page.
2. Upload the extracted `makasete-seo/` folder to your server's `/wp-content/plugins/` directory.
3. In your WordPress admin dashboard, navigate to **Plugins** and activate **Makasete SEO**.

---

## After Activation

Once the plugin is active, connect it to your [Makasete](https://makasete.app/) account using a WordPress Application Password:

1. In your WordPress admin dashboard, go to **Users → Profile** (or click your username in the top right).
2. Scroll down to the **Application Passwords** section.
3. Enter a name for the application (e.g. `Makasete`) and click **Add New Application Password**.
4. Copy the generated password — it will only be shown once.
5. In the [Makasete dashboard](https://makasete.app/), go to your site's settings and paste the application password.

Your WordPress site is now connected to the Makasete AI SEO automation platform. Articles will begin appearing on your site according to your subscription schedule.

## Updating the Plugin

Repeat the installation steps above with the latest release. Existing settings are preserved across updates.

## FAQ

**Do I need any technical skills?**
No. Connect your WordPress site once, pick a topic seed, and walk away. Articles appear on your site on schedule, automatically.

**Are the articles fact-checked?**
Yes. Every claim is cross-referenced against live web sources before publication. Anything that doesn't hold up gets rewritten until it does.

**Which languages does Makasete support?**
English and Japanese. Each WordPress site is configured for one language; articles are generated in that language and tuned to its conventions.

**Do the articles look like AI wrote them?**
No. Tone is matched to your site, voice is tuned to your audience, and the body is composed paragraph by paragraph — not from boilerplate.
